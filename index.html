使用hexo搭建blog

Posted on 2019-11-29

1. 使用hexo搭建博客

  1. 安装node.js
  • 先下载node.js。 https://nodejs.org/en/ 一直下一步安装node

  • 查看node安装是否成功

    • 管理员登录终端 cmd。
    • node -v
    • npm -v

  • 更换npm的下载源为淘宝镜像源

    • npm install -g cnpm –registry=https://registry.npm.taobao.org
  1. 安装hexo: cnpm install -g hexo-cli
  • hexo -v查看是否安装成功
  1. 使用hexo来搭建blog
  • 现在一个blog文件夹，并终端打开该文件夹。
  • hexo init - 初始化博客。需要下载git（官网下载即可，和下载node.js类似）
  • hexo s - 启动 本地4000端口
  1. 使用hexo发布文章

  • hexo n “使用hexo搭建blog” - 新建一篇文章，然后打开该文章，编辑即可

    退回到blog目录

  • hexo clean - 清理

  • hexo g - 生成

  • hexo s

2. 将搭建好的blog部署到github上

登录github，现在一个仓库

  • 新建仓库名: github昵称.github.io

继续终端安装hexo的插件

  • cnpm install –save hexo-deployer-git

设置blog目录下的_config.yml文件。在文件末尾修改

部署到远端github - 404不成功应该是 ssh密钥或者git环境变量的问题

  • hexo d
  • 然后博客的url就变成了 仓库名

3. 更换博客主题

  • git clone https://github.com/theme-next/hexo-theme-next themes/next

  • 设置blog目录下的_config.yml文件。

  • hexo clean

  • hexo g

  • hexo s

  • hexo d
